CAP 2022

Esta aplicação web que apresento é uma ferramenta de estudo pensada especialmente para aqueles que precisam passar no exame do CAP da Espanha. Ela realiza webscraping das perguntas do site do Ministério dos Transportes da Espanha, seleciona os links dos tópicos das questões, acessa cada um deles e baixa todas as perguntas, criando um arquivo com os dados brutos.
Os dados são organizados e apresentados ao usuário de forma aleatória, permitindo que você teste seus conhecimentos de forma desafiadora e eficiente. Além disso, a aplicação conta com um sistema de contagem de acertos, erros e questões não respondidas, que é realizado por meio de javascript e salva em cookies no navegador do usuário para que você não perca o seu progresso.
Ainda estou trabalhando no desenvolvimento desta ferramenta e planejamos implementar novas funcionalidades, como a organização das questões por tema e a criação de um sistema de login para controlar os usuários. Acredito que essas melhorias serão fundamentais para tornar a aplicação ainda mais útil e eficiente para aqueles que precisam se preparar para o exame do CAP.

Triac Soluções em eletrônica

O site da empresa de manutenção de eletrodomésticos foi desenvolvido com a combinação de HTML, CSS e JavaScript para garantir uma experiência de usuário otimizada e atraente.
A estrutura do site foi cuidadosamente planejada para ser intuitiva e fácil de navegar, com um menu responsivo que se adapta perfeitamente a qualquer tamanho de tela. Além disso, incluímos um campo para envio de mensagens para que você possa entrar em contato conosco de forma rápida e fácil, assim como fotos com descrições e animações para tornar a sua experiência mais agradável e interativa.
Utilizamos as tecnologias mais atualizadas para garantir que o site tenha um desempenho otimizado e rápido, e estamos sempre em busca de formas de melhorar ainda mais a experiência de navegação do usuário.
