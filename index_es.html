CAP 2022

Esta aplicación web que presento es una herramienta de estudio diseñada especialmente para aquellos que necesitan pasar el examen CAP. Realiza web scraping de las preguntas del sitio web del Ministerio de Transporte de España, selecciona los enlaces a los temas de las preguntas, accede a cada uno de ellos y descarga todas las preguntas, creando un archivo con los datos brutos.
Los datos se organizan y se presentan al usuario de forma aleatoria, lo que le permite poner a prueba sus conocimientos de manera desafiante y eficiente. Además, la aplicación cuenta con un sistema para contar las respuestas correctas, los errores y las preguntas sin responder, que se realiza mediante javascript y se guarda en cookies en el navegador del usuario para que no pierda su progreso.
Todavía estoy trabajando en el desarrollo de esta herramienta y planeamos implementar nuevas funcionalidades, como la organización de las preguntas por tema y la creación de un sistema de inicio de sesión para controlar a los usuarios. Creo que estas mejoras serán fundamentales para hacer que la aplicación sea aún más útil y eficiente para aquellos que necesitan prepararse para el examen CAP.

Triac Soluções em eletrônica

Este sitio web de la empresa de mantenimiento de electrodomésticos ha sido desarrollado con la combinación de HTML, CSS y JavaScript para garantizar una experiencia de usuario optimizada y atractiva. La estructura del sitio ha sido cuidadosamente planificada para ser intuitiva y fácil de navegar, con un menú responsivo que se adapta perfectamente a cualquier tamaño de pantalla. Además, hemos incluido un campo para el envío de mensajes para que pueda ponerse en contacto con nosotros de manera rápida y fácil, así como fotos con descripciones y animaciones para hacer su experiencia más agradable e interactiva. Utilizamos las tecnologías más actualizadas para garantizar que el sitio tenga un rendimiento optimizado y rápido, y siempre estamos buscando formas de mejorar aún más la experiencia de navegación del usuario.
